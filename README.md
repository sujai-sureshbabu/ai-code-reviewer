
# AI Code Reviewer

An AI-powered code reviewer with a user-friendly UI, built using:

- Python
- Streamlit
- Ollama
- Mistral LLM

## Features

- Automated code reviews
- Suggestions for improvements
- Simple UI for interaction

## Prerequisites

- Docker
- Ollama
- Mistral LLM

## Installation

1. Install the local Ollama LLM server:
   ```sh
   brew install ollama
2. Pull mistral
   ```sh
   ollama pull mistral

3. Start the app:
   ```sh
   docker compose up -d
